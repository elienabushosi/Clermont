## 2. Color Usage Guidelines

### Boston Blue (#4090C2)

- Use for primary CTAs, links, active states, and action-oriented icons
- Represents clarity, trust, and intelligence

### My Pink (#D59285)

- Use sparingly for highlights, secondary buttons, and emphasis
- Adds warmth and human tone
- Avoid for body text

### Cape Honey (#FEE2A9)

- Use for page backgrounds, large surfaces, empty states, and content sections
- Establishes a cozy, welcoming foundation

### Jungle Mist (#ADCCD3)

- Use for panels, secondary navigation, dividers, and informational sections
- Organizes complexity without visual noise

## 3. Supporting Neutrals (Required)

These neutrals are required for accessibility and readability:

- **Charcoal (Primary Text)**: #2E2E2C
- **Soft White (Cards / Panels)**: #FAFAF8
